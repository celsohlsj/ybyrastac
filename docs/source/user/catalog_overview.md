# Visão geral do catálogo

O catálogo YbYráSTAC está organizado em 5 temas:

| Tema | Coleções | Cobertura |
|------|---------|-----------|
| `fragmentation`  | `ybyra-mspa-ma`, `ybyra-mspa-br` | Maranhão e Brasil |
| `primary-forest` | `ybyra-primary-forest`           | Brasil (MapBiomas 10.1) |
| `emissions`      | `ybyra-emissions-brazil`         | Brasil (5 biomas) |
| `fire`           | `ybyra-fire-probability-pa`      | Pará |
| `recovery`       | `ybyra-secondary-forest-recovery`| Brasil (Chapman-Richards) |

Todas as coleções seguem o padrão STAC 1.0.0 e apontam para ativos COG
(GeoTIFFs cloud-optimized) ou Zarr, hospedados em armazenamento S3-compatível.
